Above are the three main projects made for my Digital Media Design Class.

Crows Animal Wiki is a webpage done Wikipedia style showcasing information on American Crows.

Studio Killers Website is a four page website about the band Studio Killers. It has a main page, information about the band, their discography, and a photo gallery.

New Dictionary Function is a mockup of a new function we would like to see in the dictionary.com app. This was a group project with Lauren and Shannon, and we decided to redesign the app so it was less cluttered and add a flashcard function for studying.
